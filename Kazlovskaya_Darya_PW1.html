When HTML5 has become a standard? What were the main reasons for developing new standard?
HTML5 has become a standard on 28 October 2014. The main reasons for the development of the new standard were the development and the need for new features.
What is a difference between W3C HTML5 and WHATWG HTML Living Standard?
Difference between W3C HTML5 and WHATWG HTML Living Standard is that the W3C "HTML5" standard was snapshot-based and static, while the WHATWG "HTML living standard" is continually updated.
What is the minimal HTML page code?
<!DOCTYPE html>
<html>
<head>
<title>Title</title>
</head>
<body>
<p>page content</p>
</body>
</html>
Comment on semantic and presentational markup. Why presentational markup is considered deprecated now?
Presentation HTML focuses on style and appearance, while semantic HTML focuses on conveying the structure and content of a page. Because of this, semantic HTML gets an advantage, because it is more convenient and practical.
How you can check browser support for particular tag or attribute?
You can check browser support for particular tag or attribute using Modernizr (It checks for support of HTML5 tags and attributes and provides a consistent way to handle them.)
What are the audio and video formats, officially supported by HTML standard? What happens if you use other format?
Audio and video formats officially supported by HTML standard: MP3, AAC, Ogg Vorbis, MP4, WebM, Ogg Theora. If you use other format browsers that don’t recognize the format won’t be able to play the audio/video.
